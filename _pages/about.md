## Selected Projects

### Heat Residual U-Net (HR U-Net)

**Anatomical Layer Segmentation for Chronic Low Back Pain using 3D Ultrasound**\
Funded by NIH (R61AT012282)

- Designed and implemented HR U-Net with thermal conduction modules and residual feature reconstruction
- Segmented six anatomical layers in 3D ultrasound
- Achieved **82.9% average Dice**, +1.4% improvement over SOTA (p < 0.05)

---

### Cross-Scan Fusion Network

**Registration-Based Image Synthesis Framework for Annotation-Efficient 3D Ultrasound Segmentation**
Funded by NIH (R61AT012282)
- Semi-supervised segmentation via deformation-field-based synthesis
- Projected Hausdorff loss for realistic augmentation
- Spine segmentation Dice: **92% → 94% (p < 0.05)**
- Ultrasound Dice: **78% → 81% (p < 0.01)**
- Semi-supervised learning: **+10.45% Dice improvement**

---

### Medical Report Error Diagnosis System (LLM-based)

- Multi-LLM system with knowledge base integration
- GPT-3.5 inference + GPT2 classification (LoRA fine-tuning)
- Neural re-ranking for performance optimization
- **+3% precision improvement** over baseline

---

### Adaptive Multimodal QA Learning System
